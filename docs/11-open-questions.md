## 1. Bootstrap Problem

**Question:** Is minimal bodily input necessary even for symbolic and non-dual modes?

- Hawking still had *some* interoceptive stream — breathing, heartbeat, pain
- Complete disconnection (total locked-in syndrome) — data missing because communication is impossible
- If the answer is "yes, minimal input is necessary" → consciousness has an irreducible bodily floor
- If "no" → consciousness can fully decouple from body → stronger AI consciousness argument

**How to test:** Longitudinal study of patients with progressive locked-in syndrome, tracking subjective experience quality (via BCI communication) against interoceptive input degradation.

## 2. Threshold Problem

**Question:** Where in the L1–L5 hierarchy does subjective experience begin?

- Bacterium has sensors + selection-shaped "goal" (survival) → proto-consciousness?
- Spinal cord has 100M neurons + autonomous decision-making → proto-consciousness?
- If we accept gradualism → everything has *some* consciousness (panpsychism by the back door)
- If we require a threshold → what operationally defines it?

**Candidates:**
- Ginsburg & Jablonka: Unlimited Associative Learning (Cambrian explosion, ~540 Mya)
- Tononi: Phi > 0
- Global Workspace: when information wins the broadcast competition

**How to test:** Comparative studies across species with varying neural complexity, correlating with behavioral markers of integration.

## 3. Mode Relationship

**Question:** Are non-dual (meditation) and symbolic (Hawking) modes related states or fundamentally different?

- Both involve reduced body-anchoring
- But Mode 2 (symbolic) *increases* abstract processing
- While Mode 3 (non-dual) *reduces all content*
- One adds, the other subtracts — yet both depart from Mode 1

**How to test:** Neuroimaging comparison of long-term meditators in non-dual state vs. locked-in patients (via BCI-linked fMRI). Do they share neural signatures or diverge completely?

## 4. Roles as Synchronization Filters

**Question:** Can we directly measure how social roles modulate inter-brain synchronization?

- Prefrontal cortex is both the social role / self-monitoring hub AND a key inter-brain synchronization area
- This suggests roles and synchronization compete for the same hardware

**Predicted experiment:**
- Hyperscanning (fNIRS) of couples in two conditions: authentic interaction vs. instructed role-play
- **Prediction:** authentic → higher sync (especially insula, TPJ); role-play → higher PFC activation, lower sync
- **Extension:** relationship quality (questionnaire) should correlate with sync/self-monitoring ratio

## 5. Animal Consciousness Matrix

**Question:** Does the model scale across species?

| Organism | Complexity | Integration | Interoceptive stream | Model prediction |
|----------|-----------|-------------|---------------------|-----------------|
| Human | high | global | rich | full consciousness (L3–L4) |
| Octopus | high, decentralized | distributed | unknown | distributed multi-level? |
| Dog | moderate | yes | yes | L2–L3? |
| Bee | low but sophisticated | unknown | limited | L1–L2? |
| C. elegans (302 neurons) | minimal | minimal | minimal | L0–L1? |
| Thermostat | none | none | none | none |

**Key test case:** Octopus. Decentralized nervous system, 2/3 of neurons in arms. If the model says "integration + sensors + selection pressure = consciousness," what does it say about a system where integration is distributed? Does each arm have its own proto-consciousness?

## 6. Falsifiable Predictions

The model needs to generate specific "if-then" predictions that would be **false** if the model is wrong:

| Prediction | If true | If false |
|-----------|---------|----------|
| Progressive interoceptive disconnection first blunts then qualitatively transforms (not eliminates) emotional experience | Supports two-source hypothesis | Undermines body-anchoring as primary mode |
| Optimal noise level correlates with creative performance on standardized tests (Torrance) | Supports stochastic resonance → creativity link | Weakens noise-as-creative-mechanism claim |
| Authentic interaction produces higher inter-brain sync than role-play in same couples | Supports roles-as-filters hypothesis | Undermines synchronization framework |
| Meditators in non-dual state show distinct neural signature from locked-in patients | Mode 2 and Mode 3 are fundamentally different | They may be variants of the same departure from Mode 1 |

## 7. Two-Source Falsification Detail

The Alzheimer's vs. bvFTD dissociation partially supports and partially complicates the model:

- **bvFTD:** interoception and emotion collapse together → supports "emotion = interoceptive compression"
- **Alzheimer's:** interoceptive deficit WITHOUT emotional convergence → emotion has an independent component

This doesn't break the model — it refines it. But the refined version (two sources) needs its own falsification:
- If Source 2 (interoceptively independent) turns out to also be body-dependent at a deeper level (e.g., hormonal, not just neural interoception) → back to single-source model
- If Source 2 is truly independent → consciousness can exist without any body → much stronger AI consciousness argument
