## Individual Calibration

**Genetics** set baseline weights (temperament, sensory thresholds, dopamine system).  
**Epigenetics + environment + experience** fine-tune them.  
Two people in the same situation "see" different realities — different generative models, different priors, different emotional weights.

---

## ADHD as Precision Weighting

ADHD is not a deficit — it's a different **precision weighting configuration**.

| Signal type | Neurotypical | ADHD |
|------------|-------------|------|
| Novelty | moderate precision | **high** precision |
| Routine | moderate precision | **low** precision |
| Emotional threshold | lower (subtle signals pass) | **higher** (only intense signals pass) |
| Pattern recognition | standard | **enhanced** (more anomaly detection) |
| Sustained attention | stable | **unstable** (depends on interest/novelty) |

- Dopamine modulates precision. Low dopamine = high threshold = need stronger input for signal to reach consciousness.
- "Need for extremes" = emotional threshold set higher than neurotypical. Normal emotional signals don't pass the filter. Only intense ones break through.
- "Always rushing, walking fast, driving fast" = elevated psychomotor activation maintains sufficient sensory flow. Slowing down = reduced input = brain seeks stimulation elsewhere (anxiety, restless thoughts).

### Medication Prediction

Concerta/Medikinet should **lower the threshold** → subtler emotional signals pass → less need for extremes. This is not personality change — it's recalibrating the filter.

**Tier 2** — consistent with Lawson et al. (2017) predictive processing model of ADHD.

## Inter-Generational Transfer

Calibration isn't just individual. It transfers across generations through two mechanisms:

### Cultural Evolution

- Norms, roles, expectations shape which signals are **systematically suppressed** across generations
- Example: historical suppression of female autonomy → generative model calibrated to "my needs = irrelevant signal" → prediction error from own needs systematically suppressed across generations
- This is a generalization — enormous inter-individual variability. Cultural pattern ≠ individual determination.

### Epigenetic Transfer

- Stress responses transfer transgenerationally (Yehuda et al. — Holocaust survivor descendants show altered cortisol regulation)
- Chronic suppression of autonomy may leave epigenetic traces — methylation patterns, histone modifications
- Not deterministic — sets predisposition, not destiny

**Tier 2** — epigenetic transgenerational transfer proven for stress responses, application to broader calibration patterns is extrapolation.

## Self-Worth Calibration

- **Early mirroring worked** → stable self-model → less dependence on external validation
- **Early mirroring failed** → unstable self-model → chronic need for external confirmation
- "A person moves themselves forward through an internal 'I have value and I want people to value me'" = internal model calibration. The drive for external validation is a calibration signal, not a flaw.
