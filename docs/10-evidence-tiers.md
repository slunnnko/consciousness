# Evidence Tiers

[← Back to README](../README.md)

---

## Classification System

- **Tier 1:** Strong evidence — supported by meta-analyses, replicated experiments, or established neuroscience frameworks
- **Tier 2:** Consistent but unproven — logically coherent with Tier 1 findings, supported by indirect evidence, but lacks direct experimental confirmation
- **Tier 3:** Speculative — interesting interpretations, internally consistent with the model, but not testable with current methods or based on extrapolation
