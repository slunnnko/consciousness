## The Cycle

```
Perception → Emotional tag → Interpretation → Model update → Changed perception
     ↑                                                              |
     └──────────────────── Prediction error ←───────────────────────┘
```

The brain doesn't wait for input, then react. It **predicts** what input should look like, compares with actual input, and processes only the **difference** (prediction error). This minimizes energy expenditure.

- **Emotions are fast heuristic tags** — they classify a situation before conscious processing. Amygdala response: ~12ms. Conscious awareness: ~300–500ms.
- The model either **updates** (learning) or the organism **changes the environment** to fit the model (action). Both minimize prediction error.
- This is Karl Friston's **Active Inference** framework.

**Tier 1.**

## Memory as Emotional Reconstruction

Memory is not file retrieval. It is **reconstruction from an emotional tag**.

- The brain stores emotional traces, not data files
- Each recall reconstructs the memory from the emotional tag — potentially modifying it (Nader, 2003 — memory reconsolidation)
- This is why false memories exist — reconstruction from emotion, not from data
- Weak delayed recall (e.g., 11th percentile in ADHD) ≠ missing data. It means the **cognitive retrieval path** is weak.
- Emotional/bodily retrieval paths (music, rhythm, melody) bypass the cognitive deficit — they use a different channel

### First-Person Evidence

> "Every thought and memory is an emotion. It's encoded in emotion. Even in my case — the emotion is quieter, but it's an emotion, a feeling."

This directly supports Damasio's somatic marker hypothesis and Barrett's constructed emotion theory as the fundamental encoding layer.

**Tier 1** (emotion as encoding) / **Tier 2** (application to ADHD memory deficit).

## Pattern Recognition and Anomaly Detection

> "I use ears more than eyes. I see and hear patterns, then I ignore them and focus on deviations."

This is a first-person description of predictive processing:

1. Brain generates prediction (pattern)
2. Compares with input
3. Processes only **prediction error** (deviation)

### ADHD Configuration

- Routine patterns = low precision (boring, ignored)
- Deviations = high precision (interesting, focus)
- Result: strong pattern recognition, weak sustained attention on predictable input

### Auditory Preference

- Auditory processing has higher temporal precision than visual
- Detecting deviations in temporal patterns (rhythm, intonation, prosody) is faster through hearing
- Explains both the music-as-memory hack and the preference for auditory channel

## Noise as Creative Mechanism

Random sounds and unexpected percepts enable connections between otherwise separate neural networks.

- **Stochastic resonance** — proven phenomenon where small amounts of noise improve detection of weak signals (Moss et al., 2004). **Tier 1.**
- Extrapolation: noise bridges networks that don't normally communicate → basis of creativity. **Tier 2** — consistent but unproven leap from signal detection to network bridging.
- **Default Mode Network** — when "doing nothing," brain connects areas that don't communicate during focused work. Best ideas in shower, while walking, while falling asleep.
- **Psychedelics** do this pharmacologically — reduce prior precision → areas that normally don't communicate get connected → kaleidoscopic perception, synesthesia, "everything is connected"

### Optimal Noise Level

Too little noise → no bridging, stale patterns.  
Too much noise → overwhelm, sensory chaos.  
Optimal = somewhere between silence and chaos.

**ADHD relevance:** ADHD brain has naturally weaker filters. More "random" signals get through. Advantage: more bridging, more unexpected connections. Cost: overwhelm, inability to filter irrelevant input. White noise as a regulation tool = **controlled dose of randomness**.
